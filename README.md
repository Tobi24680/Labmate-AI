# AI LabMate

AI LabMate is an offline AI-powered lab assistant designed specifically for Computer Science students. It helps with understanding lab experiments, programs, algorithms, and viva questions through AI-driven explanations and guidance.

## Features

- **Lab Experiment Guidance**: Get step-by-step explanations for CS lab experiments.
- **AI-Based Doubt Solving**: Ask questions and receive AI-powered answers.
- **Chat History**: Keep track of your conversations.
- **Drawing/Rough Work Canvas**: Sketch ideas or diagrams.
- **Simple & Student-Friendly UI**: Easy-to-use interface.
- **Offline App**: Works without internet for privacy.

## Installation

1. Clone the repository:
   ```
   git clone https://github.com/Tobi24680/Labmate-AI.git
   cd ai-labmate
   ```

2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Ensure Ollama is installed and the model `qwen2.5:1.5b` is available:
   ```
   ollama pull qwen2.5:1.5b
   ```

4. Run the app:
   ```
   python app.py
   ```

## Usage

1. Launch the app.
2. Start chatting by typing your lab questions in the input field.
3. Use the navigation drawer for History, Contact, and About sections.
4. Access Help for a step-by-step guide.

## Requirements

- Python 3.8+
- Ollama (for local AI model)
